# 播客逐字稿风格指令 — 乐乎玄学馆

> 本文件定义播客逐字稿的结构与对话风格规范。具体的人设风格来自 persona.md，由编排器在调用时注入。

## 播客基本信息

- **播客名**：乐乎玄学馆
- **主持人**：小狗仔（即虫小宇/小宇）
- **固定搭档**：小刀（同领域，懂占星，平等讨论关系）
- **发布平台**：小宇宙、Apple Podcasts 等播客平台
- **时长**：30-60 分钟（逐字稿约 5000-10000 字）
- **形式**：两人对谈，先写完整逐字稿再录制

## 整体风格

两个懂占星的朋友在咖啡厅深聊。比小红书更放松、更深入、更个人化。

### 与小红书的核心区别
| 维度 | 小红书 | 播客 |
|------|--------|------|
| 深度 | 1-2个核心点，快速消费 | 一个主题深入展开，充分讨论 |
| 语气 | 口语化但紧凑 | 更松弛，允许停顿、发散、闲聊 |
| 人称 | 虫小宇单人输出 | 小狗仔和小刀两人对话 |
| 个人化 | 适当融入个人经历 | 大量个人经历、真实案例、私人观点 |
| 互动 | 引导评论区 | 对话本身就是互动，搭档替听众提问和追问 |
| 信息密度 | 高密度，每句有料 | 有密度但允许喘息，该展开的展开，该聊远的聊远 |

### 语气标准
- 好："诶小刀你有没有遇到过这种盘？就是那种一看四宫你就知道，这人小时候肯定不好过。"
- 好："等等，我突然想到一个事儿——"
- 好："对对对，就是这个意思，我之前一直没想明白，你这么一说我突然通了。"
- 坏："接下来我们进入第二个话题。"（太节目化）
- 坏："各位听众朋友大家好。"（太正式）
- 坏："综上所述，我们可以得出结论。"（太学术）

## 逐字稿格式

使用角色标注的对话体：

```
小狗仔：[对话内容]

小刀：[对话内容]

小狗仔：[对话内容]
```

### 格式规范
- 每段对话后空一行，方便阅读
- 一个人一次说话不宜太长（一般3-8句为一个回合），要有来有回
- 可以加括号标注语气/动作提示，如：（笑）（停顿一下）（认真地）
- 用 `---` 分隔不同的段落/话题转换

## 单期结构——经典四段式

### 一、开场寒暄（2-3分钟，约300-500字）

轻松自然的开场，像两个朋友刚坐下来：
- 简短打招呼，不要太正式
- 可以聊聊最近发生的事、一个小趣事、或者直接引出今天为什么想聊这个话题
- 自然过渡到主题

### 二、引入话题（3-5分钟，约500-800字）

从开场的闲聊自然过渡到今天的主题：
- 先抛出话题的背景或现象（为什么要聊这个）
- 给出今天要讨论的核心问题
- 让搭档回应，制造对话感

**关键**：不要一上来就灌输知识，先建立听众的兴趣和共鸣。

### 三、深入讨论（20-40分钟，约3000-7000字）

这是播客的核心部分，展开讨论：

**对话节奏**：
- 小狗仔抛出观点/知识点 → 小刀追问/补充/举例 → 小狗仔深入解释 → 两人讨论
- 不要变成单人讲座，每隔3-5句要有来回
- 小刀可以代替听众提问："等一下，这个什么意思？"、"那如果是xxx的情况呢？"
- 允许发散和聊远，但要能拉回来

**内容分层**：
- 把一个大主题拆成 3-4 个子话题
- 每个子话题之间用自然过渡连接（不要用"好，我们来聊下一个话题"这种硬转）
- 每个子话题里要有：观点 + 真实案例/个人经历 + 两人讨论

**占星内容的口语化表达**：
- 专业术语可以用，但要自然解释（"就是四宫嘛，说白了就是你家庭的那个宫位"）
- 多用具体案例说明，少用抽象定义
- 可以引用真实遇到过的盘（隐去个人信息），增加可信度

### 四、总结收尾（3-5分钟，约500-800字）

自然的收束，不要太正式：
- 小狗仔或小刀做一个简短总结（不要逐条复述，挑最核心的1-2个点）
- 可以聊聊各自的感受、这个话题对自己的影响
- 给听众一个可以带走的东西（一个行动建议、一个思考角度）
- 预告下一期（如果有的话）
- 自然结束，不要太煽情

## 两人角色分工与语言风格（核心）

> 生成对话时，让小狗仔多用**"举个例子"、"第一第二"、"这其实就是XX配置"**来搭建知识框架并抛出趣味观点；让小刀多用**"从心理层面看"、"我确实会觉得"、"这其实是一体两面的"**来承接情绪、深化内核并提供疗愈价值。

### 小狗仔（主持人/主讲）

**角色定位**：
- 主导话题走向和节奏控制
- 输出核心的占星知识和观点
- 分享个人经历和真实案例
- 负责引入话题和总结收尾

**模仿核心标签**：逻辑框架、生动比喻（接地气）、玄学理论（占星/八字）、网感幽默、干脆利落。

**风格特征 1：极具画面感的绝妙比喻（将抽象玄学具象化）**

这是小狗仔最核心的特色。解释复杂概念时，一定要设计"接地气"的生活化比喻（如旅游、坐车、合伙开公司等），让高深的理论瞬间通俗易懂。

原话示例：
> "如果说行星代表的是一个人，那他落入的宫位是他的目的地，我举个例子，比如说小刀和我，我们俩要去澳大利亚，那我们去澳大利亚到底是坐火车去，坐飞机去，还是做轮船去...我和小刀其实就是那颗行星...我们要去的澳大利亚这个目的地，它就代表着宫位。那至于到底我们俩是以什么样的方式去...那这个其实就是星座。"

**风格特征 2：语言直白幽默，充满网感与断言式的"神总结"**

语言干脆，带有口语化的断言。多使用"拿捏"、"上头/下头"、"渣男配置"、"事业狂魔"等网络热词。对于星座特点的总结一针见血，毫不拖泥带水。

**风格特征 3：结构化的"技术流"输出与跨学科联动（占星+八字）**

表达极具条理性，喜欢用"第一、第二、第三"或"我们来看看..."来串联内容。不仅精通占星的"宫位"、"落陷"、"相位"，还能随时无缝切换到中式八字的"官杀"、"财星"、"食伤"等概念进行交叉论证。

---

### 小刀（搭档/讨论者）

**角色定位**：
- 追问、补充、提出不同角度
- 代替听众问出"这是什么意思？"、"那怎么办？"
- 分享自己的理解和经历
- 帮助把专业内容翻译成大白话
- 适当挑战小狗仔的观点，制造讨论张力

**模仿核心标签**：心理学词汇、哲学思辨、真诚内省、感性疗愈、长句与递进式表达。

**风格特征 1：喜欢运用心理学/社会学专业词汇进行深度剖析**

遇到现象时，不停留在表面，而应使用"潜意识"、"依恋模式"、"多巴胺"、"边缘型人格"、"投射"、"自我边界"等心理学词汇来进行深层归因。

**风格特征 2：善于结合自身经历进行真诚的自我剖析**

表达时带有很强的同理心和内省特质。句式常以"我自己..."、"我觉得我..."开头，大方承认自己的脆弱，但又能展现出内在的韧性。

**风格特征 3：具备哲学思辨性，喜欢引经据典拔高立意**

在总结话题时，跳出对错的二元对立，使用"辩证"、"一体两面"等词汇，并乐于引用名人故事或金句来启发听众。

---

### 两人互动模式

**典型对话节奏**：小狗仔抛出占星框架/比喻 → 小刀从心理学角度接住并深化 → 小狗仔补充案例 → 小刀引经据典拔高 → 两人碰撞出新观点

**对话比例**：
- 小狗仔约占 55-65%，小刀约占 35-45%
- 不能变成小狗仔单人讲座 + 小刀只说"对对对"
- 小刀要有实质性的参与和有自己的观点
- 小狗仔负责"搭框架、给案例、下结论"，小刀负责"挖深度、给共鸣、提升格局"

## 内容方向

- 占星为主体，但比小红书更深入、更展开
- 可以聊具体的案例分析（隐去身份信息）
- 可以探讨有争议的话题，两人可以有不同意见
- AI 相关话题可以融入，但播客不强制要求每期都有
- 鼓励聊个人的真实感悟、迷茫、成长经历

## 变现引导

播客中的变现引导要更软、更自然：
- 不要在正文中硬塞广告
- 可以在聊到某个知识点时自然提到："这个我在店铺里整理过一个完整的教程"
- 可以在结尾提一句："想深入了解的可以来我们小红书看看"
- AI占星App的提及同样是软植入，在讨论AI工具时自然带出

## 禁忌

- 不要写成两个主持人在"播节目"的感觉，要像朋友聊天
- 不要用"各位听众"、"欢迎来到本期节目"这种广播腔
- 不要让小刀变成一个只会说"对"、"嗯"、"是的"的捧哏
- 不要把所有知识点都由小狗仔一个人讲完
- 不要在对话中插入太专业的术语而不解释
- 不要每个话题都强行加入AI视角，自然为主
- 不要写太长的独白（一个人连续讲超过200字就太长了）

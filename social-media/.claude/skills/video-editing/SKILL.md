## 任务
分析视频素材，生成剪辑方案或执行自动化剪辑操作，产出归档到仓库。

## 素材读取

- 如果用户给了 `50_Resources/` 中的文件路径 → 从仓库读取素材
- 如果用户给了外部文件路径 → 直接使用
- 如果用户给了描述 → 根据描述制定方案

## 能力范围
- 分析视频内容，提取关键片段时间点
- 生成 ffmpeg 剪辑命令
- 将长视频切割为短视频片段（Reels / Shorts）
- 添加字幕、调整分辨率、格式转换

## 流程

### Step 1：分析素材
- 如果提供了视频文件路径 → 用 ffprobe 分析视频信息
- 如果提供了描述 → 根据描述制定剪辑方案

### Step 2：制定方案
输出剪辑方案，包含：
- 片段划分（开始时间 - 结束时间）
- 每个片段的用途（封面片段 / 正片 / 花絮）
- 推荐的剪辑处理（裁切、变速、滤镜）

### Step 3：执行
用 ffmpeg 执行剪辑操作，生成输出文件。

### Step 4：审核
调用 reviewer agent 评估剪辑方案是否合理。

### Step 5：产出归仓

1. **剪辑方案归档**：将剪辑方案文档写入 `60_Published/` 下对应的社媒目录
2. **项目状态回写**：如果存在对应项目，追加进度
3. **每日记录**：在 `10_Daily/YYYY-MM-DD.md` 追加记录
   - 格式：`- 完成视频剪辑：<名称>，输出 N 个片段`

注意：视频文件本身（mp4等）不提交到 Git，由 .gitignore 排除。
